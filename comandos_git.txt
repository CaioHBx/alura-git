git init (comando para inicializar em um projeto)

git status (comando para visualizar informações)

git config --local user.name "Caio"
git config --local user.email "caio.batista@bsoft.email"

git add "arquivo" (adiciona o arquivo mencionado para a monitoração)

git add . (adiciona o todos os arquivos do diretorio para a monitoração)

git commit -m "mensagem sobre o commit" (salva as alterações do arquivo e manda um mensagem)

git log (mostra o historico do git)

git log --oneline (mostra o historico do git em uma linha por objeto) 

git log -p (mostra o historico do git completo)
